Driver Feedback Loop

State A: Live Shift Minimalist view only. Progress bar, current pace, forecast badge. No charts while driving.
State B: Pit Stop Analytics Post-shift deep dive. Altair pace chart (actual vs. ideal), safety log with severity filtering.
Note: The Data Processing / Cleaning step (marked "⚠ Assumed", dashed border) is not implemented in the current prototype. All sensor data is synthetically generated in a clean, pre-formatted state via generate_synthetic_data.py, so null handling, outlier filtering, and format normalisation are not required. All other stages, including all four Stage 1 sources, Sensor Fusion, Data Sampling, both engines, and the Streamlit dashboard are fully implemented.
